# Mass Effect — STE Physics Ontology

A first-principles physics ontology built on a single substrate: **STE** (Space-Time Energy).

## Core Premise

The universe is a finite bubble of self-attracting potential fluid. Every observable phenomenon — mass, gravity, heat, inertia — emerges from one rule applied at different scales and measured from different frames.
